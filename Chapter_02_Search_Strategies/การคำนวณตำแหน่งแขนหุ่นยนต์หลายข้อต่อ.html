การคำนวณตำแหน่งแขนหุ่นยนต์หลายข้อต่อ

แขนหุ่นยนต์ 1 ข้อต่อ (1 DOF)

หลักการ: แขนหุ่นยนต์ 1 ข้อต่อจะมีการหมุนรอบจุดฐาน ตำแหน่งปลายแขนจะเป็นวงกลม
สูตรการคำนวณ:
x = L₁ × cos(θ₁)
y = L₁ × sin(θ₁)

แขนหุ่นยนต์ 2 ข้อต่อ (2 DOF)

หลักการ: แขนหุ่นยนต์ 2 ข้อต่อจะมีการหมุนที่ข้อต่อฐานและข้อต่อที่สอง ทำให้มีความยืดหยุ่นมากขึ้น
สูตรการคำนวณ:
ข้อต่อที่ 1:
x₁ = L₁ × cos(θ₁)
y₁ = L₁ × sin(θ₁)
ตำแหน่งปลายแขน:
x₂ = L₁ × cos(θ₁) + L₂ × cos(θ₁ + θ₂)
y₂ = L₁ × sin(θ₁) + L₂ × sin(θ₁ + θ₂)

แขนหุ่นยนต์ 3 ข้อต่อ (3 DOF)

หลักการ: แขนหุ่นยนต์ 3 ข้อต่อจะมีการหมุนที่ข้อต่อฐาน ข้อต่อที่สอง และข้อต่อที่สาม ทำให้สามารถเข้าถึงตำแหน่งได้หลากหลายมากขึ้น
สูตรการคำนวณ:
ข้อต่อที่ 1:
x₁ = L₁ × cos(θ₁)
y₁ = L₁ × sin(θ₁)
ข้อต่อที่ 2:
x₂ = L₁ × cos(θ₁) + L₂ × cos(θ₁ + θ₂)
y₂ = L₁ × sin(θ₁) + L₂ × sin(θ₁ + θ₂)
ตำแหน่งปลายแขน:
x₃ = x₂ + L₃ × cos(θ₁ + θ₂ + θ₃)
y₃ = y₂ + L₃ × sin(θ₁ + θ₂ + θ₃)

แขนหุ่นยนต์ 3 ข้อต่อ พร้อมตำแหน่งเป้าหมาย

หลักการ: คำนวณตำแหน่งปลายแขน (End-effector) และหาระยะห่างจากตำแหน่งเป้าหมาย (Target) ที่ต้องการ

ตำแหน่งเป้าหมาย

สูตรการคำนวณ:
ตำแหน่งปลายแขน (x₃, y₃): คำนวณเหมือนแขน 3 ข้อต่อ
ระยะห่าง (Distance):
d = √((targetX - x₃)² + (targetY - y₃)²)

แขนหุ่นยนต์ 3 ข้อต่อ พร้อมเป้าหมายและอุปสรรค

หลักการ: คำนวณตำแหน่งปลายแขน, ระยะห่างถึงเป้าหมาย, และระยะห่างจากแขนแต่ละท่อนถึงอุปสรรค
หลักการคำนวณระยะห่างจากแขนถึงอุปสรรค (Vector Projection)

การคำนวณหาระยะที่สั้นที่สุดจาก ศูนย์กลางของอุปสรรค (จุด P) ไปยัง แขนแต่ละท่อน (ส่วนของเส้นตรง AB)

1. กำหนดจุด:

  • อุปสรรค: P(pₓ, pᵧ)
  • ข้อต่อเริ่มต้น: A(aₓ, aᵧ)
  • ข้อต่อสิ้นสุด: B(bₓ, bᵧ)

2. คำนวณค่า t:
ค่า t คือสัดส่วนที่บอกว่าจุดตั้งฉากอยู่ ณ ตำแหน่งใดบนเส้นตรง AB

t = ((pₓ - aₓ)(bₓ - aₓ) + (pᵧ - aᵧ)(bᵧ - aᵧ)) / ((bₓ - aₓ)² + (bᵧ - aᵧ)²)

3. ตรวจสอบช่วงของ t:

  • ถ้า t < 0: จุดที่ใกล้ที่สุดคือ A
  • ถ้า t > 1: จุดที่ใกล้ที่สุดคือ B
  • ถ้า 0 ≤ t ≤ 1: จุดที่ใกล้ที่สุดอยู่บนเส้น AB

เราสามารถปรับค่า t ให้อยู่ในช่วง [0, 1] (หากเกินช่วงก็ปรับให้เป็นค่าสุดขอบนี้) เพื่อให้ได้จุดที่ใกล้ที่สุดที่อยู่บน ส่วนของเส้นตรง AB เท่านั้น

4. หาพิกัดจุดที่ใกล้ที่สุด C(cₓ, cᵧ):

cₓ = aₓ + t ⋅ (bₓ - aₓ)

cᵧ = aᵧ + t ⋅ (bᵧ - aᵧ)

5. คำนวณระยะทางสุดท้าย (Distance):
คือระยะห่างระหว่างจุด P และ C

Distance = √((pₓ - cₓ)² + (pᵧ - cᵧ)²)

การควบคุมแขนหุ่นยนต์

สูตรการคำนวณ:
- ระยะห่างถึงเป้าหมาย: d = √((targetX - x₃)² + (targetY - y₃)²)
- ระยะห่างถึงอุปสรรค: คำนวณระยะทางที่สั้นที่สุดจากจุด (Obstacle Center) ไปยังเส้นแขนแต่ละท่อน
